Explanation

This covers the most common ways of displaying a PDF in a webpage. The PDF can be viewed inline in two different ways, opened in a separate tab, or easily downloaded for future reference. In most browsers, all four methods behave as expected. If we were to replace the PDF with some other file that Firefox can display, like an MP4, all four methods would behave as expected in FirefoxIn current versions of Firefox, it is impossible to make this webpage fully functional.

If PDF behavior is set to "Open in Firefox", the embed, iframe, and direct link behave correctly, but clicking the download link will download the file to the temp folder (without opening a download dialogue) and open it in a new tab.
